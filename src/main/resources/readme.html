SWRLTab

Protégé Desktop plugin for working with the SWRL rule and SQWRL query languages. The plugin provides two tabs: the SWRLTab, which can be used to manage SWRL rules, and the SQWRLTab, which works with SQWRL queries. The tabs can be activated via the "Windows > Tabs..." menu item.

Please refer to the wiki for detailed documentation.
